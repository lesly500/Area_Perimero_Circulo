# PROGRAMA#1
programa para calcular el area y el perimetro de un circulo radio r

# Analisis


## Input
### Variabls de entrada
r: radio del circulo
### processing
p: perimetro del circulo
p=2πr

a: area del circulo
a=πrr

### Output
a,p
